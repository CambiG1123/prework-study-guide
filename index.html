HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style webpages with CSS, inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The most common method is an external css style sheet. This allows the developer to make changes to their CSS rules within a separate file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Example git flow: navigate to file of current project, (git checkout -b feature/new-branch), make changes and save in vs code, (git add -A), (git commit -m "made changes"), (git pull origin main), if up to date, then (git push origin feature/new-branch). Open github and merge pull request.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Use an assignment operator = to assign data to a variable
  • ie. var shape = "circle";
  • Strings: var topics = "HTML, CSS, Git, JavaScript"; Notice that strings are surrounded by quotations ' " '
  • Arrays: var shapes = ["triangle", "square", "pentagon", "circle"]; A single variable that contains a group of data
